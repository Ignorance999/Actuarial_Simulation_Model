Documentation¶

  1. Brief Explanation of GenTable
    1. This class is inherited from Table
    2. It will read XML lookup files.

    3. Other comments This name is from my software, which may be changed in the future. It will read XML in “test_input” folder. Currently, it will recognize those columns that can be regarded as “keys” and other columns as “values”. The “BODY” member is a dictionary. I am thinking of changing this self.BODY to pandas.dataframe instead.
